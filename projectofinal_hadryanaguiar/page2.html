Processos de crianção:

O processo de criação do Mod é relativamente simples, deve ter-se primeiro noção de Java básico e Java do Minecraft básico. É necessário a instalação do Java 17 para configurar o mods da versão 1.20.1 (Minha versão do MOD) O Mod foi criado pelo programa MCreator em Forge, os modelos das entidades e blocos 3D estão sendo modelados pelo BlockBench, junto com as texturas, as texturas pixeladas de itens básicos do Minecraft estão sendo criadas pelo Piskel.

Inspirações:

O Mod está sendo inspirado completamente no Modpack: Orespawn, com o objetivo de ser um Mod grande recheado de itens/objetivos/blocos/entidades e entre outros só que para versões mais recentes.
